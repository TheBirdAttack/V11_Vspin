# V11_Vspin
Reverse Engineered V11 VSpin python driver

Make sure to use an RS232 adapter with a genuine FTDI chip and full hardware handshaking (RTS/CTS).

Backend from https://github.com/PyLabRobot/pylabrobot
Some adjustments for the old Velocity 11 Firmware
